# Yetrealm Issue Tracker

This repository is the official issue tracker for **Yetrealm**, a Minecraft survival progression mod.

Please use this repository to report:

- Bugs
- Crashes
- Compatibility issues
- Performance problems
- Balance problems
- Feature suggestions

## Download

CurseForge: https://www.curseforge.com/minecraft/mc-mods/yetrealm  
Modrinth: https://modrinth.com/mod/yetrealm

## Before Reporting a Bug

Please make sure you are using the latest version of Yetrealm.

Also check whether the issue has already been reported in the Issues tab.

## Bug Reports Should Include

When reporting a bug or crash, please include:

- Minecraft version
- NeoForge version
- Yetrealm version
- Singleplayer or dedicated server
- Full mod list
- `latest.log`
- Crash report, if the game crashed
- Steps to reproduce the issue
- Screenshots or videos, if useful

## Important

Issues without logs, crash reports, or clear reproduction steps may be difficult to investigate.

Thank you for helping improve Yetrealm.
